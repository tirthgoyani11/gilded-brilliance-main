# Vmora Luxury Jewelry Platform

Premium storefront for diamond and silver jewelry with a dedicated loose-diamond marketplace experience.

## Core Experience

- Luxury homepage with cinematic hero, category curation, diamond search preview, best sellers, education, and brand story
- Jewelry storefront and loose-diamond marketplace navigation
- Premium design system with gold/black/white direction and typography tuned for luxury branding

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui primitives
- Framer Motion for interactions
- Vitest + Playwright for testing

## Local Development

```sh
npm install
npm run dev
```

The app runs on `http://localhost:8080` by default.

## Quality Checks

```sh
npm run lint
npm run test
npm run build
```

## Brand Notes

- Brand name: Vmora
- Visual language: minimal, elegant, white space first
- Color palette: luxury gold, deep black, pure white, and neutral diamond gray
